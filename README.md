# ASTRA PRO

🚀 SMS Automation Tool for Android devices

## Features
- ✓ Bulk SMS sending
- ✓ Multi-device support
- ✓ OTP verification
- ✓ Number filtering
- ✓ Automatic version check

## Installation

Download the latest version from [Releases](https://github.com/sabbirislamzilanji0007-svg/astra-pro/releases)

1. Download ASTRA_PRO.exe
2. Run the executable
3. Connect Android device via ADB/LDPlayer
4. Add phone numbers to numbers.txt
5. Start the tool

## System Requirements

- Windows 7 or higher
- LDPlayer 4.1+ or Android emulator
- ADB connection support
- Python 3.8+ (if running from source)

## Version Check

This tool automatically checks for updates from GitHub.
When a new version is released:
- Old versions will show an update notification
- You will need to download the latest version

Current Version: 1.1

## Files

- version.json - Version check configuration
- requirements.txt - Python dependencies
- .gitignore - Git ignore rules
